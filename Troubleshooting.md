# `ng` is not running under windows

In den Angular Beispielen (Speziell dem aufsetzen von einem neuen Projekt) kam es zu Problemen mit der globalen installation von `ng`. 

Windows blockiert die Ausführung von skripten. 

- Angular Dokumentation: https://angular.dev/tools/cli/setup-local#powershell-execution-policy
- Stackoverflow Lösungen https://stackoverflow.com/questions/58032631/why-powershell-does-not-run-angular-commands/58044573#58044573

# nextjs / angular is not starting with cryptic error

Bitte stellen Sie sicher, dass Sie alle Code Beispiele in einem Ordner ausführen welcher:

- Keine leerzeichen beinhaltet
- Keine Sonderzeichen beinhaltet
- Nicht auf einen Cloud Ordner zeigt (z.B. OneDrive, Dropbox, Google Drive)

Im besten Fall erstellen Sie sich einen Code Ordner unter ihrem Benutzer Verzeichnis und kopieren das Repo hier hinein. Für die Zusammenarbeit mit anderen nutzen Sie Git in Kombination mit Github (Für Studenten ist Github Pro sogar kostenlos)
