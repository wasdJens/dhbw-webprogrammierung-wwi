# DHBW Web Programming NestJS Starter Template

- Includes TypeORM with Better SQLite3 as database
- Basic CRUD operations for Products (As an sample)
- Validation Pipe enabled
- API Versioning enabled (URI Versioning)
- CORS enabled for local development

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start:debug
```
